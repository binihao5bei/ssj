脚本用途：
用来自动卸载安装输入法，避免人工频繁操作的成本

本次更新：
1.最新版（install_new）去掉了按照分支的脚本下载，需要手动填写分支路径
2.自定义版（install_zdy）取消区分分支路径下载的方式，需要手动填写下载路径

脚本介绍：
1.总控脚本：install-king.sh
更新：仅区分安装最新版本、自定义版本以及单纯的卸载输入法(Uninstall.sh)
2.可以通过总控脚本选择执行，也可以通过单独脚本执行

总控脚本使用说明：

sh install-king.sh后弹出菜单，用户自行选择：
a.安装最新输入法
b.安装自定义输入法
c.仅卸载当前输入法

其他按键均会退出

ps：
1.机器必须联网，安装curl和wget
安装命令：sudo apt install curl ； sudo apt-get install wget
2.对于最新版的安装，脚本可以判断系统架构，并通过接口获取最新下载链接，但需要手动填写下载分支，填写规则如下：
develop-2.1，develop-2.2，develop-2.3，develop-2.4，develop（目前代表2.5，以后可以代表2.6等）
3.对于自定义安装，需要在对应脚本里修改，进入脚本后找到根据自身系统的架构找到对应链接位置，替换链接，再运行脚本即可（务必先替换链接再运行脚本！！！）
4.涉及到密码部分必须自行修改脚本中的输入密码部分，脚本中输入密码命令是：echo 你的密码 | sudo（yum） -S xxx
